# M1.7 单 cycle 探索 — 把 M1.1-M1.6 串起来

## 目标

> 在某个时刻，给定当前 OctoMap M、全局图 G、当前关节构型 θ_now，跑**一个完整 cycle**：采样候选 → 评分 → 选最优 → 执行第一段 → 更新 G。

这是 Phase 1 的"心脏"。M1.8 在它外面包一层 loop + dead-end recovery。

## 在管线中的位置

```text
执行入口:
  while not phase_switch_triggered:
    cycle_result = run_one_cycle(state)        ← M1.7 在这里
    state = apply(cycle_result, state)
```

## 单 cycle 7 步流程

```text
                   ┌─────────────────────────────────────────────┐
                   │ Input: theta_now, M (voxel map), G (graph) │
                   └────────────────────┬────────────────────────┘
                                        ▼
       Step 1  在末端笛卡尔小立方体内撒 N 个候选 ee_pose
                                        ▼
       Step 2  IK 反推关节角, 失败的丢弃
                                        ▼
       Step 3  对每个候选的关节角做整臂碰撞检测, 撞的丢弃
                                        ▼
       Step 4  对剩下的算 gain (M1.6) + score = gain - λ·cost
                                        ▼
       Step 5  argmax score → v_best
                                        ▼
       Step 6  把 v_best 加入 G, 加边 (theta_now, v_best)
                                        ▼
       Step 7  返回 v_best 作为下一段执行目标
                                        ▼
              ┌───────────────────────────────────┐
              │ 调用方执行 v_best, 拍 depth, 整合 │
              │ 进 voxel map                       │
              └───────────────────────────────────┘
```

## 关键设计决定

| 决定 | 选项 | 选 | 理由 |
|------|------|------|------|
| 采样空间 | 关节空间 vs 笛卡尔 | 笛卡尔 + IK | 直观控制相机运动方向 |
| 采样形状 | 球壳 vs 立方体 | 立方体 | 简单；GBPlanner2 也用 box |
| 采样大小 | 0.3 m vs 0.5 m | 0.3 m | 工件小 |
| 朝向决定 | 随机 vs 朝目标 | 朝目标 + 抖动 | 加速收敛；GBPlanner2/WG-NBVP 都这么做 |
| 候选数 N | 50 vs 200 | 100 | 算 IK + 碰撞 + gain 总开销 ~1 s |
| 评分权重 λ | 距离惩罚系数 | 0.3 | 经验值，调试时调 |
| 路径决策 | 单步 vs 多步 | 单步 | 简单；M1.8 处理多步 |
| 切换检测 | 每 cycle vs 每帧 | 每 cycle | 简单；正确性够 |

## 实施步骤

### Step 1: sample_candidate_ee_poses

```python
def sample_candidate_ee_poses(ee_pos_now, p_weld_center, cfg, n):
    # 位置: 立方体均匀采样
    half = cfg.sample_box_half
    rng = np.random.default_rng()
    pos_offsets = rng.uniform(-half, half, size=(n, 3))
    positions = ee_pos_now + pos_offsets
    
    # 朝向: 大致朝 p_weld, 加抖动
    poses = np.zeros((n, 4, 4))
    for i, pos in enumerate(positions):
        # 主朝向: 指向 p_weld
        z_axis = p_weld_center - pos
        z_axis /= np.linalg.norm(z_axis)
        # 加 yaw/pitch 抖动
        yaw = rng.uniform(-np.radians(cfg.yaw_jitter_deg), np.radians(cfg.yaw_jitter_deg))
        pitch = rng.uniform(-np.radians(cfg.pitch_jitter_deg), np.radians(cfg.pitch_jitter_deg))
        z_axis = rotate_vector(z_axis, [0, 0, 1], yaw)
        z_axis = rotate_vector(z_axis, [1, 0, 0], pitch)
        # up = world z (朝上), 然后正交化
        up = np.array([0, 0, 1])
        x_axis = np.cross(up, z_axis); x_axis /= np.linalg.norm(x_axis)
        y_axis = np.cross(z_axis, x_axis)
        R = np.stack([x_axis, y_axis, z_axis], axis=1)
        poses[i, :3, :3] = R
        poses[i, :3, 3] = pos
        poses[i, 3, 3] = 1
    return poses
```

### Step 4: 评分

```python
def score_candidates(poses, thetas, theta_now, voxel_map, p_weld_center, arm_kin, cfg):
    K = len(poses)
    gain_cfg = GainConfig(
        w_vol=cfg.w_vol, w_target=cfg.w_target,
        n_rays_h=8, n_rays_v=8,
        max_range=cfg.voxel_max_range,
    )
    
    scores = torch.zeros(K, device='cuda')
    for i in range(K):
        T = poses[i]
        cam_pos = T[:3, 3]
        cam_dir = T[:3, 2]
        cam_up  = -T[:3, 1]
        
        g = compute_gain(cam_pos, cam_dir, cam_up, voxel_map, p_weld_center, gain_cfg)
        # 距离 cost
        cost = float(np.linalg.norm(thetas[i].cpu().numpy() - theta_now))
        scores[i] = g - cfg.lambda_cost * cost
    return scores
```

### Step 5-7: 主流程

```python
def run_one_cycle(theta_now, voxel_map, graph, arm_kin,
                  seam_points, seam_tangents, p_weld_center,
                  cycle_idx, cfg, intrinsics):
    # 当前 EE 位姿
    fk = arm_kin.fk(torch.tensor(theta_now).unsqueeze(0).cuda())
    ee_pos_now = fk['ee_pos_world'][0].cpu().numpy()
    
    # Step 1: 采样
    candidate_poses = sample_candidate_ee_poses(
        ee_pos_now, p_weld_center, cfg, cfg.n_samples,
    )
    n_total = len(candidate_poses)
    
    # Step 2: IK
    poses_after_ik, thetas_after_ik = filter_by_ik(candidate_poses, arm_kin)
    n_after_ik = len(poses_after_ik)
    
    # Step 3: 碰撞
    poses_after_col, thetas_after_col = filter_by_collision(
        thetas_after_ik, poses_after_ik, arm_kin, voxel_map,
    )
    n_after_col = len(poses_after_col)
    
    if n_after_col == 0:
        return CycleResult(success=False, ...)
    
    # Step 4: 评分
    scores = score_candidates(poses_after_col, thetas_after_col, theta_now,
                              voxel_map, p_weld_center, arm_kin, cfg)
    
    # Step 5: 最优
    best_idx = int(scores.argmax().item())
    best_score = float(scores[best_idx])
    best_pose = poses_after_col[best_idx]
    best_theta = thetas_after_col[best_idx].cpu().numpy()
    
    # Step 6: 加入图
    new_node = GraphNode(
        node_id=str(uuid.uuid4()),
        theta=best_theta,
        ee_pos_world=best_pose[:3, 3],
        ee_dir_world=best_pose[:3, 2],
        cycle_added=cycle_idx,
        last_gain=best_score,
    )
    graph.add_node(new_node)
    
    # 加边: 从 nearest 现有节点到 new_node (实际上是 theta_now → new_node)
    # 假设当前 theta_now 已经在 graph 里了 (cycle 0 的初始化负责)
    nearest = graph.nearest_node(theta_now)
    if nearest is not None:
        edge_cost = float(np.linalg.norm(theta_now - nearest.theta))
        # 这条边已经验证过 (因为 v_best 的碰撞检测过, 而它和 nearest 同样靠近)
        # 严格说应该再验证一次 nearest → new_node, 但成本高, 这里偷懒
        # 改进: 只为 theta_now 加边, 其他靠 M1.8
        graph.add_edge(nearest.node_id, new_node.node_id, weight=edge_cost)
    
    # Step 7: 检测 phase switch
    cur_vp = Viewpoint(
        pos=best_pose[:3, 3],
        dir=best_pose[:3, 2],
        up=-best_pose[:3, 1],
    )
    p_observed = detect_phase_switch(
        voxel_map, seam_points, seam_tangents, cur_vp, cfg, intrinsics,
    )
    
    return CycleResult(
        success=True,
        next_theta=best_theta,
        next_ee_pose_world=best_pose,
        best_score=best_score,
        best_gain=best_score,  # 没分开就用一致
        best_node_id=new_node.node_id,
        n_candidates_total=n_total,
        n_candidates_after_ik=n_after_ik,
        n_candidates_after_collision=n_after_col,
        phase_switch_triggered=len(p_observed) > 0,
        p_seam_observed=p_observed,
    )
```
