# M1.1 RaycastSim — 用 USD/OBJ 网格模拟深度相机

## 目标

> 给一个相机位姿 (SE(3))，输出一张深度图（每个像素到场景表面的距离），不依赖 Isaac Sim 渲染。

用途：

- M1.2 体素栅格的输入（喂"假深度"建图，避免 Isaac Sim 启动慢）
- 单元测试可重复（同一相机位姿一定出同一张图）
- 单帧 < 10 ms（vs Isaac Sim 渲染单帧 ~50 ms 起）

## 在管线中的位置

```text
RaycastSim.render(pose) ─→ depth (h, w)
                            │
                            ▼
                        VoxelMap.integrate()  (M1.2)
```

后期 M1.9 会再加一个 `IsaacSimDepth` 实现，两者可互相替换（用一个抽象接口 `DepthSource`）。

## 关键设计决定

| 决定 | 选项 | 选 | 理由 |
|------|------|------|------|
| 几何源 | USD vs OBJ | OBJ | 已有 `.obj` 在 pkl 同目录；USD 还得起 Isaac Sim 才能解析 |
| 加速结构 | BVH (`embreex`) vs 暴力 | embreex | 已装；3060 上 720p 单帧 < 5 ms |
| 输出 device | CPU vs GPU | CPU 出 + 转 GPU | embreex CPU only；后续 GPU op 自己转 |
| 深度截断 | inf vs 固定值 (2 m) | inf | 截断逻辑放在 M1.2 里，本模块只给真值 |
| 多 mesh 处理 | 单一 mesh | 多 mesh + 拼接 | 工件 + 障碍物可能多个 OBJ |

### 相机模型（OpenCV/Isaac Sim 约定）

```text
camera frame:
    +z 朝前 (光轴)
    +x 朝右
    +y 朝下

像素 (u, v) 对应方向 (在 camera frame):
    x_c = (u - cx) / fx
    y_c = (v - cy) / fy
    z_c = 1
    dir = normalize([x_c, y_c, z_c])

world 中的射线方向:
    R_world_cam = camera_pose[:3, :3]
    dir_world = R_world_cam @ dir_cam
    origin_world = camera_pose[:3, 3]
```

## 实施步骤

### Step 1: 加载 + 合并 mesh

```python
import trimesh

mesh_list = [trimesh.load(p, force='mesh') for p in mesh_paths]
# 应用每个 mesh 的初始 pose
for m, pose in zip(mesh_list, mesh_poses):
    m.apply_transform(pose)
# 合并
combined = trimesh.util.concatenate(mesh_list)
# 建 embree BVH
intersector = trimesh.ray.ray_pyembree.RayMeshIntersector(combined)
```

### Step 2: 生成像素射线

```python
H, W = intrin.height, intrin.width
u, v = np.meshgrid(np.arange(W), np.arange(H))
x_c = (u - intrin.cx) / intrin.fx     # (H, W)
y_c = (v - intrin.cy) / intrin.fy
dirs_cam = np.stack([x_c, y_c, np.ones_like(x_c)], -1)  # (H, W, 3)
dirs_cam /= np.linalg.norm(dirs_cam, axis=-1, keepdims=True)
```

### Step 3: 把射线变换到世界

```python
R = camera_pose[:3, :3]
t = camera_pose[:3, 3]
dirs_world = (R @ dirs_cam.reshape(-1, 3).T).T   # (H*W, 3)
origins_world = np.broadcast_to(t, dirs_world.shape).copy()
```

### Step 4: 求交

```python
locations, ray_idx, _ = intersector.intersects_location(
    ray_origins=origins_world,
    ray_directions=dirs_world,
    multiple_hits=False,
)
# embreex 只返回 hit 的射线; ray_idx 给出哪些像素 hit 了
depth_flat = np.zeros(H * W, dtype=np.float32)
hit_dist = np.linalg.norm(locations - origins_world[ray_idx], axis=1)
depth_flat[ray_idx] = hit_dist
depth = depth_flat.reshape(H, W)
return torch.from_numpy(depth).cuda()
```

### Step 5: 性能优化（可选，先把功能跑通再说）

- batch render: 多个 pose 一次 ray pool
- numpy → torch zero-copy
- 内置 RayMeshIntersector 已多线程
