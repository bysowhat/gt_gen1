### Step 2: 视频录制

Isaac Sim 自带 `omni.kit.capture.viewport`，但 Python 调用比较复杂。简单方案：用 `ffmpeg` 抓 Isaac Sim 窗口。

```python
class MovieRecorder:
    def __init__(self, output_path):
        import subprocess
        # 抓 Isaac Sim 主窗口 (X11)
        self.proc = subprocess.Popen([
            'ffmpeg', '-y',
            '-f', 'x11grab', '-framerate', '30',
            '-video_size', '1920x1080',
            '-i', ':0.0',
            '-c:v', 'libx264', '-preset', 'ultrafast',
            output_path,
        ])
    
    def stop(self):
        self.proc.terminate()
        self.proc.wait()
```

进阶方案：用 Isaac Sim 自己的 `RecordingViewport`（更稳定，但 API 复杂）。Phase 1 先用 ffmpeg。

    Path(dir).mkdir(parents=True, exist_ok=True)

        img = render_cycle_summary(c)        # matplotlib 拼图

`render_cycle_summary` 画 4 子图：

- 左上：当前体素地图三视图（XY 投影，红=occupied，绿=free）
- 右上：候选分布（散点，颜色 = score）
- 左下：score 排行榜 top 10
- 右下：cycle 指标表（n_total, n_after_ik, n_after_collision, best_score）

class Phase1Visualizer:
    def __init__(self, env: IsaacSimEnv):
        self.env = env
        self._ids = []          # 已添加的 marker prim 路径
    
    def update(self, state: Phase1State):
        # 清掉旧 marker
        for pid in self._ids:
            self.env.remove_prim(pid)
        self._ids = []
        
        # 画 graph 节点
        for node in state.graph._g.nodes(data='data'):
            ee = node[1].ee_pos_world
            color = (0.2, 0.4, 1.0) if node[1].last_gain < 5 else (1.0, 0.5, 0.0)
            pid = self.env.add_marker(f"node_{node[0]}", ee, color, radius=0.005)
