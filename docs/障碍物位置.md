# Isaac Sim 焊缝观测避障场景构建

本文档记录如何在 Isaac Sim 中布置障碍物，使机械臂必须在避障的情况下观测工件焊缝的指定位置。

## 问题目标

基础场景包括：

- 机械臂
- 工件
- 工件上的焊缝目标点
- 安装在机械臂末端或指定位置的相机

目标不是简单地在场景中增加障碍物，而是让障碍物满足以下效果：

- 机械臂默认观测路径被阻挡
- 默认相机视线被遮挡，或默认机械臂运动轨迹发生碰撞
- 机械臂仍然可以通过绕行、改变关节构型或改变观察角度看到焊缝指定位置
- 场景不能变成完全无解

## 障碍物放置的核心原则

障碍物应放在两类关键区域中：

1. 机械臂从初始姿态到默认观测姿态的运动通道附近
2. 相机从默认观测位姿看向焊缝目标点的视线通道附近

也就是说，障碍物应放在机械臂“最想走的路”或相机“最想看的视线”上，而不是随意放在场景中。

## 推荐放置策略

### 1. 挡住默认最优观测位姿

先在没有障碍物的情况下，让机械臂规划一次焊缝观测任务，得到默认相机观测位姿：

```text
T_cam_best
```

然后将障碍物放在默认观测位姿附近，例如默认相机位姿的前方或侧前方 10–30 cm。

这种布置会使原本最短、最舒服的观测姿态不可用，迫使机械臂换一个观察角度。

俯视示意：

```text
          障碍物
            ███
            ███
默认相机位姿 X  →  焊缝点 P
                    工件
机械臂基座 R
```

适用目标：

- 构造“默认观测姿态不可用”的场景
- 测试机械臂是否能找到替代观测角度
- 验证观测位姿搜索能力

### 2. 放在默认相机视线通道中

如果希望机械臂因为“看不见焊缝”而调整姿态，障碍物应放在默认相机位置和焊缝目标点之间。

设：

```text
C_best = 默认相机位置
P_weld = 焊缝目标点
```

障碍物中心可以按以下方式放置：

```text
O = P_weld + α * (C_best - P_weld)
```

其中：

```text
α = 0.3 ~ 0.7
```

即障碍物位于焊缝点到默认相机位置连线的 30%–70% 处。

示意：

```text
默认相机 C_best  ---- 障碍物 O ---- 焊缝点 P_weld
```

建议不要让障碍物完全贴近焊缝点，否则可能导致所有可行视角都被遮挡。

适用目标：

- 构造“默认视线被遮挡”的场景
- 验证相机可见性约束
- 迫使机械臂从侧面、上方或斜向重新寻找观察位姿

### 3. 放在机械臂关节扫掠空间中

机械臂避障不应只考虑末端。很多情况下真正容易碰撞的是：

- 肘部
- 前臂
- 腕部
- 相机支架

可以先在无障碍条件下规划一条默认轨迹，然后记录各个 link 的运动轨迹，将障碍物放在肘部、前臂或腕部经过最多的区域。

俯视示意：

```text
机械臂基座 R
    \
     \   肘部默认轨迹
      \  o o o o o
          ███
          ███ 障碍物
              \
               相机观测位姿
```

这种方法可以迫使机械臂改变整体构型，例如抬肘、绕侧面或改变腕部姿态。

适用目标：

- 验证全身避障能力
- 避免只测试末端避障
- 构造更接近真实工装环境的机械臂运动约束

## 推荐的初始障碍物布置方案

如果需要快速搭建一个有效测试场景，推荐使用如下方案：

```text
障碍物放在焊缝目标点和默认相机观测点之间的 40%–60% 位置，
再向焊缝法向的侧向偏移 5–10 cm，
高度覆盖默认相机视线高度，
形状使用长条挡板。
```

侧视示意：

```text
        默认相机 C_best
             \
              \
               █████  障碍物挡板
                \
                 \
                焊缝点 P_weld
              工件表面
```

推荐初始尺寸：

```text
长度：0.3–0.6 m
宽度：0.05–0.2 m
高度：0.3–0.8 m
```

该方案通常会迫使机械臂：

- 从正面观察改为侧面观察
- 或从低位观察改为高位俯视
- 同时在运动过程中避开挡板

## 不建议的障碍物位置

### 1. 不要紧贴焊缝

如果障碍物紧贴焊缝目标点，可能导致所有视角都无法看到目标点，任务直接无解。

### 2. 不要放得离机械臂基座太近

靠近机械臂基座的障碍物会大范围限制机械臂可达空间，容易把问题变成单纯的运动学不可达，而不是“避障后可观测”。

### 3. 不要放在远离默认路径的位置

如果障碍物不在默认运动通道或默认视线通道附近，机械臂可能完全不需要绕行，无法体现避障效果。

## 判断障碍物是否放置合适

一个合适的障碍物布置应满足以下条件：

```text
1. 无障碍条件下的默认最优观测姿态不可用
2. 默认相机视线被遮挡，或默认机械臂路径发生碰撞
3. 至少仍存在一个侧向、上方或斜向观测姿态可行
4. 机械臂为了到达新的观测姿态，需要明显绕开障碍物
```

如果不满足第 1、2 点，障碍物太弱，机械臂不需要避障。

如果不满足第 3 点，障碍物太强，场景可能无解。

如果不满足第 4 点，障碍物虽然遮挡了局部视线，但没有真正构造出机械臂避障需求。

## 建议的场景构建流程

```text
1. 在无障碍场景中规划机械臂到焊缝观测位姿
2. 记录默认相机位置 C_best、焊缝目标点 P_weld 和默认机械臂轨迹
3. 在 C_best 到 P_weld 的连线上放置一个长条挡板
4. 将挡板向侧向偏移 5–10 cm，避免完全堵死所有视角
5. 检查默认视线是否被遮挡
6. 检查是否仍然存在侧向、上方或斜向可见视角
7. 如果机械臂仍然不需要明显绕行，将障碍物移向肘部或前臂扫掠路径
8. 如果任务无解，减小障碍物尺寸或远离焊缝目标点
```

## 总结

障碍物的最佳位置不是焊缝点本身，也不是场景中的随机位置，而是：

```text
默认观测视线通道 + 默认机械臂运动通道 + 关键 link 扫掠空间
```

优先从默认相机视线通道中布置长条挡板，再根据机械臂实际轨迹调整到肘部、前臂或腕部的扫掠区域。这样最容易构造出“机械臂必须避障才能看到焊缝指定位置”的有效仿真场景。
