# 障碍物 3D 结构

本文档记录 Isaac Sim 焊缝观测避障场景中可使用的障碍物 3D 结构类型。目标是构造既能造成机械臂运动阻挡，又能造成相机视线遮挡，但仍保留可行观测通道的工业场景。

## 1. 钢板 / 挡板类

### 单块钢板

形状：

```text
薄长方体
```

特点：

- 厚度小
- 面积大
- 容易遮挡相机视线
- 适合模拟隔板、防护板、工装挡板

典型尺寸：

```text
长：0.4–1.5 m
宽：0.2–0.8 m
厚：0.005–0.05 m
```

适合放在：

```text
相机默认观测点与焊缝之间
```

### 倾斜钢板

形状：

```text
一块斜放的薄板
```

示意：

```text
   /
  /  钢板
 /
```

特点：

- 比垂直钢板更真实
- 会产生斜向遮挡
- 迫使机械臂从侧上方或侧下方观察

### L 型钢板

形状：

```text
两块钢板垂直连接
```

示意：

```text
└
```

特点：

- 同时阻挡两个方向
- 适合模拟夹具、挡边、支撑角钢
- 比单块板更容易形成绕行需求

### U 型钢板槽

形状：

```text
三块钢板组成 U 型槽
```

示意：

```text
┌   ┐
│   │
└───┘
```

特点：

- 工件可以放在槽内
- 焊缝部分暴露
- 机械臂需要从开口方向或上方观察

## 2. 多平面开口盒 / 五面体

五面体可以理解为缺一个面的盒子，适合模拟半封闭焊接空间。

形状：

```text
底面 + 左面 + 右面 + 后面 + 顶面
前面开口
```

示意：

```text
前方开口

┌────────┐
│        │
│ 工件   │
│        │
└────────┘
```

特点：

- 工件在半封闭空间中
- 机械臂不能从任意方向接近
- 相机只能从开口、侧缝或局部窗口观察
- 适合模拟箱体内部焊缝、壳体焊缝、腔体内焊缝

常见变体：

### 顶部开口盒

形状：

```text
无顶面
```

适合机械臂从上方探入观察。

### 侧面开口盒

形状：

```text
无左面或无右面
```

适合构造侧向观察需求。

### 带观察窗口的盒体

形状：

```text
五面体 + 小窗口
```

特点：

- 默认大视角被挡住
- 只有特定角度能看到焊缝
- 适合测试可见性约束

## 3. 圆柱钢管类

### 单根圆柱管

形状：

```text
长圆柱体
```

特点：

- 模拟管线、支撑杆、横梁
- 会造成局部遮挡
- 对机械臂 link 碰撞影响明显

典型尺寸：

```text
长度：0.5–2.0 m
半径：0.02–0.15 m
```

适合放置：

```text
横跨相机视线通道
```

示意：

```text
相机 ----  OOOOO  ---- 焊缝
```

### 多根平行钢管

形状：

```text
多根圆柱并排
```

示意：

```text
OOOOOOOO
OOOOOOOO
OOOOOOOO
```

特点：

- 类似管束、护栏、支架
- 会形成多条狭窄观察通道
- 机械臂需要选择从管缝中观察或绕过

### 交叉钢管

形状：

```text
两根或多根圆柱交叉
```

示意：

```text
 X
```

特点：

- 适合模拟临时支撑、管架、脚手架
- 遮挡不规则
- 会让路径规划更复杂

### 弯管 / U 型管

形状：

```text
弯曲圆柱
```

示意：

```text
∪
```

特点：

- 比直管更接近真实管路
- 既有线状遮挡，也有包围趋势
- 适合放在工件附近形成绕行约束

## 4. 方管 / 矩形管 / 三角管

### 方管

形状：

```text
长方体梁
```

示意：

```text
████████
```

特点：

- 比圆管碰撞边界更明显
- 适合模拟工装横梁、方钢架、机架

### 矩形管框架

形状：

```text
多个方管组成框架
```

示意：

```text
┌────────┐
│        │
└────────┘
```

特点：

- 中间有孔洞
- 可以让相机从孔中看焊缝
- 机械臂本体可能必须绕过框架

### 三角管 / 三角支架

形状：

```text
三角形截面管
或三根杆组成三角框
```

示意：

```text
  /\
 /__\
```

特点：

- 稳定支撑结构常见
- 会形成斜向边界
- 比水平或垂直障碍更容易迫使机械臂改变姿态

## 5. 工装夹具类

### 压紧夹具

形状：

```text
底座 + 压板 + 螺杆/圆柱
```

特点：

- 真实焊接场景常见
- 常靠近工件
- 会遮挡焊缝局部区域
- 对末端相机和腕部避障影响明显

### 定位块

形状：

```text
小型方块 / 楔形块
```

特点：

- 尺寸不大，但贴近工件
- 会局部遮挡焊缝
- 适合测试精细避障

### V 型支撑块

形状：

```text
V 型槽
```

示意：

```text
\  /
 \/
```

特点：

- 适合管件或圆柱工件定位
- 焊缝可能位于 V 型槽附近
- 机械臂需要从槽口方向观察

## 6. 框架 / 门架 / 支架类

### 门型框架

形状：

```text
两根竖梁 + 一根横梁
```

示意：

```text
┌─────┐
│     │
│     │
```

特点：

- 中间可通视
- 横梁和立柱会限制机械臂进入
- 适合测试机械臂是否能从窗口中观察

### 矩形笼式框架

形状：

```text
多个方管组成半包围框架
```

特点：

- 工件在框架内部
- 焊缝可见通道有限
- 适合构造复杂避障观测任务

### 斜撑框架

形状：

```text
矩形框架 + 斜向支撑杆
```

示意：

```text
┌─────┐
│ \   │
│  \  │
└─────┘
```

特点：

- 斜撑会破坏简单的直线路径
- 比纯矩形框架更容易产生绕行需求

## 7. 不规则遮挡件

### 楔形块

形状：

```text
三棱柱 / 斜面块
```

特点：

- 产生斜向遮挡
- 适合模拟坡口、支撑垫块、导向块

### 台阶形障碍物

形状：

```text
多个长方体叠加
```

示意：

```text
██
████
██████
```

特点：

- 遮挡高度变化明显
- 适合测试机械臂从不同高度寻找视角

### 多面折弯板

形状：

```text
折线形钢板
```

示意：

```text
__/‾‾\__
```

特点：

- 比单钢板更真实
- 可能同时挡住多个角度
- 适合模拟折弯钣金件、防护罩、复杂夹具

## 8. 推荐优先使用的结构

如果目标是快速构造“机械臂必须避障才能看到焊缝”的场景，建议优先使用：

```text
1. 长条钢板 / 倾斜钢板
2. 五面开口盒，工件放在盒内
3. 一根或多根横向圆柱钢管
4. 方管矩形框架
5. L 型或 U 型钢板夹具
6. 三角支架 / 斜撑框架
```

这些结构容易同时造成：

- 视线遮挡
- 运动路径阻挡
- 保留可行观察通道

推荐组合：

```text
五面开口盒 + 横向圆管
```

或：

```text
方管框架 + 斜撑杆
```

这两类组合更接近真实工装环境，也更适合测试焊缝观测中的避障规划能力。
